학생 명렬표

등록된 명렬표 없음 — 기본 학교 명렬표 또는 반·번호 목록을 사용합니다.

학번(4자리) 열 또는 반·번호·이름 열을 인식합니다. B열=학번, C열=이름 형식도 지원합니다.

설치 안내에 따라 스크립트를 배포한 뒤 URL을 붙여 넣으세요.

응답 저장

GitHub 저장은 Apps Script 스크립트 속성 GITHUB_TOKEN 이 필요합니다. 설정 안내
